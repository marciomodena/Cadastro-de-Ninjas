Projeto simples desenvolvido em **Java 17** com **Spring Boot** e **Spring Web**, com o objetivo de praticar a criação de uma API REST para cadastro de ninjas.

## 🚀 Tecnologias utilizadas

- Java 17
- Spring Boot
- Spring Web
- Maven ou Gradle
- IntelliJ IDEA

## 📌 Objetivo do projeto

Criar uma API para realizar operações básicas de cadastro de ninjas, como:

- Cadastrar ninja
- Listar ninjas
- Buscar ninja por ID
- Atualizar ninja
- Deletar ninja
